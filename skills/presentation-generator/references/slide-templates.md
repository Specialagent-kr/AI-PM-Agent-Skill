# 슬라이드 유형별 상세 템플릿

## 공통 슬라이드 패턴

### 타이틀 슬라이드
```
[중앙 정렬]
프로젝트명 (40-44pt, Bold, 딥 블루)
서브타이틀 (24-28pt, 회색)
날짜 · 발표자 (18pt, 연회색)
```

### 섹션 구분 슬라이드
```
[중앙 정렬, 딥 블루 배경]
섹션명 (44pt, Bold, 흰색)
섹션 소개 1-2줄 (20pt, 연파란색)
```

### 컨텐츠 슬라이드 (기본)
```
[슬라이드 제목] (32-36pt, Bold, 상단)

• 핵심 포인트 1
• 핵심 포인트 2
• 핵심 포인트 3

[우측 또는 하단: 관련 시각 자료]
```
