## 타이포그래피

### 폰트 계층
| 레벨 | 크기 | 스타일 | 색상 |
|------|------|--------|------|
| 슬라이드 제목 | 32-36pt | Bold | 딥 블루 |
| 섹션 헤딩 | 24-28pt | Semi-bold | 딥 블루 |
| 본문 1단계 | 18-20pt | Regular | 진회색 |
| 본문 2단계 | 16-18pt | Regular | 중간 회색 |
| 캡션/주석 | 14pt | Light | 중간 회색 |
| 타이틀 슬라이드 | 40-44pt | Bold | 딥 블루/흰색 |

### 텍스트 가이드라인
- 글머리 기호당 최대 6단어
- 슬라이드당 최대 5-6개 글머리
- 제목은 Sentence case (전체 대문자 지양)
- 제목: 가운데 정렬 / 본문: 왼쪽 정렬
- 줄 간격: 1.5배

## 레이아웃 원칙

### 여백 (EMU 단위)
- 사방 여백: 457200 EMU (약 1인치 / 72pt)
- 제목과 본문 사이: 228600 EMU
- 글머리 사이: 114300-152400 EMU

### Google Slides EMU 좌표 참조
- 슬라이드 전체: 9144000 (가로) × 5143500 (세로)
- 여백 포함 콘텐츠 영역: 약 914400에서 시작

### 정렬 기준
- 모든 글머리의 왼쪽 마진 통일
- 표의 숫자는 오른쪽 정렬
- 차트와 이미지는 가운데 정렬

### 공백 원칙
- 슬라이드당 50-60% 여백 유지
- 텍스트 왼쪽, 시각 자료 오른쪽 (또는 반대)
- 슬라이드 혼잡 금지

## 슬라이드 유형별 레이아웃 가이드

### 1. 타이틀 슬라이드
```
배경: 흰색 또는 딥 블루 그라디언트
위치: 중앙 정렬
- 프로젝트명: 40-44pt, Bold
- 부제목: 24-28pt
- 날짜/발표자: 18pt
- 상단 또는 하단에 얇은 파란 선 액센트
```

### 2. 콘텐츠 슬라이드
```
상단: 슬라이드 제목 (32-36pt)
중간: 글머리 목록 (18-20pt)
    • 포인트 1
    • 포인트 2
    • 포인트 3
우측 (선택): 아이콘 또는 간단한 다이어그램
하단 좌측: 슬라이드 번호 (12pt)
```

### 3. 데이터/수치 슬라이드
```
좌측 60%: 맥락 설명 텍스트
우측 40%: 대형 수치 (오렌지/딥 블루, 36-48pt)
하단: 데이터 출처 (12pt, 회색)
```

### 4. 표 슬라이드
```
제목: 상단 (32pt)
표:
- 헤더: 딥 블루 배경, 흰색 텍스트, Bold
- 짝수 행: 연회색 배경
- 홀수 행: 흰색 배경
- 테두리: 연회색 (#e0e0e0)
```

### 5. 섹션 구분 슬라이드
```
배경: 딥 블루 (#1a237e) 전체
텍스트: 가운데 정렬
- 섹션 번호: 20pt, 연파란색
- 섹션명: 44pt, Bold, 흰색
- 설명: 20pt, 연파란색
```
