# North Star Metric (NSM)

## 개요

North Star Metric은 제품이 고객에게 전달하는 핵심 가치를 가장 잘 포착하는 단일 지표다.

**특징:**
- 고객 가치를 나타냄 (매출 아님)
- 장기 성공의 예측 지표
- 팀이 영향을 미칠 수 있음
- 이해하기 쉽고 기억하기 좋음
- 선행 지표 (후행 지표 아님)

---

## 제품 유형별 NSM 예시

| 제품 | North Star Metric | 이유 |
|------|-------------------|------|
| Spotify | 사용자당 청취 시간 | 참여도와 가치 반영 |
| Airbnb | 예약 숙박 수 | 핵심 거래, 양방향 가치 |
| Slack | 팀이 보낸 메시지 수 | 팀 협업 활동 |
| Netflix | 사용자당 시청 시간 | 콘텐츠 참여도 |
| Dropbox | 저장된 파일 수 | 스토리지 사용 = 가치 |

---

## NSM 선택 방법

**1단계 — 핵심 가치 파악:**
- 제품이 고객에게 어떤 가치를 전달하는가?
- 고객이 제품으로 무엇을 달성하는가?
- "아하 모먼트"는 무엇인가?

**2단계 — 후보 지표 브레인스토밍:**
- 고객 가치를 반영하는 지표 목록 작성
- 참여도, 사용량, 성과 지표 고려

**3단계 — 기준으로 평가:**
- ✅ 고객 가치를 나타내는가?
- ✅ 매출과 성장을 예측하는가?
- ✅ 팀이 영향을 미칠 수 있는가?
- ✅ 단순하고 이해하기 쉬운가?
- ✅ 선행 지표인가?

**4단계 — 하나의 주요 지표 선택:**
모든 기준에 가장 잘 맞는 지표 선택 후 팀 정렬

**5단계 — 지원 지표 정의:**
NSM을 이끄는 입력 지표 파악 → 지표 트리 생성

---

## North Star 프레임워크

```
North Star Metric
      │
      ├─ 입력 지표 1 (예: 신규 획득)
      ├─ 입력 지표 2 (예: 활성화)
      ├─ 입력 지표 3 (예: 참여도)
      └─ 입력 지표 4 (예: 유지율)
```

**Spotify 예시:**
```
North Star: 사용자당 청취 시간
      │
      ├─ 신규 가입자 수 (획득)
      ├─ 플레이리스트 생성 사용자 (활성화)
      ├─ 세션당 재생 곡 수 (참여도)
      └─ 주간 활성 사용자 (유지율)
```

---

## 활용 시점

✅ 적합:
- 전체 조직을 고객 가치에 정렬
- 장기 제품 성공 측정
- 제품 전략과 로드맵 방향 설정
- 크로스 팀 기능 단위 정렬
