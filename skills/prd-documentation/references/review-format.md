## 피드백 작성 원칙

**구체적으로:** PRD의 특정 섹션 또는 내용을 직접 인용하여 참조한다.

**건설적으로:** 개인 비판이 아닌 내용과 아이디어에 집중하며, 제안 또는 질문 형식으로 제시한다.

**맥락 있게:** 제안의 근거를 설명하고, 비즈니스 또는 사용자 임팩트와 연결한다.

**우선순위화:** "반드시 수정"과 "좋으면 수정" 구분하고, 임팩트가 높은 개선에 집중한다.

**실행 가능하게:** 대안 또는 해결책을 제시하며, 개선을 위한 구체적 단계를 포함한다.
